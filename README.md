## Project Overview
This project focuses on analyzing e-commerce sales data to identify trends, performance metrics, and business insights using a dashboard.

## Tools Used
- Excel 

## Key KPIs
- Total Sales
- Total Orders
- Average Order Value
- Customer Count
- Target Achievement %

## Insights
- Sales performance varies significantly across regions
- Certain customer segments contribute higher revenue
- KPI tracking helps identify gaps against targets
